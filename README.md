# Fedora L10n

Fedora Weblate translation status viewer.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install fedora-l10n
```

### Fedora/RPM
```bash
sudo dnf install fedora-l10n
```
